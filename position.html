15. Position : Sticky

11 - Introduction

La propriété CSS position permet de contôler la manière dont un élément est positionné dans le document.

Elle détermine si un élément suit le flux normal du document ou si il est placé précisément avec les propiétés top, right, bottom et left.


12. Position: Statique

Comportement: L'élément suit le flux normal position par défaut de tous les éléments HTML

top, right, bottom et left n'ont aucun effet si l'élément à cette propriété


Exemple :


                
                .static{
                    height: 50px;
                    width: 50px;
                    background-color: blue;
                    position: static;
                }

Je suis collant !!

12. Position : Relative

Comportement: L'élément est positionné par rapport à sa position normale

Il occupe toujours sa place dans le flux du document

Les propriétés top, right, bottom et left, le déplace visuellement, sans changer sa place d'origine 

                
                .relative{
                     height: 50px;
                     width: 50px;
                     background-color: blue;
                     position: relative;
                    top: 2rem;
                    left: 3rem;
                }

13. Position : Absolue

L'élément est retiré du flux

Il est postionné par rapport au premier parent positionné (non statique)

Si aucun parent n'est positionné, il se basera sur la balise "html"

15. Position : Sticky

Le comportement est un mixte entre relative et fixed

L'élément reste dans le flux jusqu'à ce qu'on scroll au point où il devient fixe
