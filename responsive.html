Responsive Design

1 - Qu'est-ce que le Responsive Design

Le Responsive Design (ou Design Adapatatif est la traduction en français) est une approche de conception web qui permet à un site de s'adapter automatiquement à la taille de l'écran.

"ctrl+mj+m" pour mettre le menu adaptatif (pour choisir la dimension de l'écran de chaque appareil)

2 - Les breakpoints

Un breakpoint est un seuil de largeur d'écran à partir du quel on modifie l'apparence de la page web afin de l'adapter au spport (mobile, tablette, desktop)

21 - Liste des breakpoints

  • Mobile => max-width: 480px
  • Tablette => max-width: 768px
  • Laptop(ordinateur portable) => max-width: 1024px
  • Desktop (les tours PC) => max-width: 1025px

22 - Attention !!!

La résolution des écrans en CSS est représenté en pixel logique.

Dans la réalité les appareils sont en pixel physiques.

Par exemple un Iphone 13 à une résolution logique de 398 x 844 pixels mais en réalité elle à une résolution physique de 2532 x 1170px.

Par exemple pour un écran Retina 3 pixels physqiues pour 1 pixel logique.

3 - Principes fondamentaux du Responsive Design

Utilisation des % ou vh/vw au lieu de valeurs fixes comme pixel(px).

Utiliser des propriétés tel que max-width: 100% pour que les images ne dépassent jamais leur conteneur

Les média queries => Ce sont des instructions qui aplliquent des styles en fonction de la taille de l'écran.

4 - Les flexbox

L'utilisation du wrap est maintenant très importante.

Actuellement nous vivons dans un monde ou les téléphones portables sont omniprésents, il est conseillé maintenant de concevoir son site en MOBILE FIRST.

Le mobile first est un concept de web design optimisé pour le mobile, il consiste à concevoir un site en mettant la propriété sur la version mobile et ensuite d'adapter le site progressivement sur des écrans plus larges
